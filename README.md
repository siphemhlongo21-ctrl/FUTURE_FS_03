# FUTURE_FS_03 – Maison Dorée Full-Stack Website

## Overview
This project is a full-stack restaurant website built as part of my internship Task 3.

It demonstrates frontend, backend, authentication, and database integration using modern technologies.

---

## Tech Stack

### Frontend
- React.js (Vite)
- HTML5
- CSS3
- JavaScript (ES6+)
- React Router DOM
- Axios
- React Hot Toast

### Backend
- Node.js
- Express.js
- JWT Authentication
- bcrypt
- dotenv
- CORS

### Database
- Microsoft SQL Server
- MSSQL driver

## How To Run

### Backend
cd server  
npm install  
npm run dev  

### Frontend
cd client  
npm install
